# AQI---An-Applicative-and-Predictive-Model

This project is a web-based Air Quality Index (AQI) monitoring system developed as a capstone project by undergraduate students of Indian Institute of Technology Patna (IIT Patna). It aims to provide a simple, interactive, and informative platform for users to understand air pollution levels across major Indian cities.

The system displays real-time AQI data, tracks major pollutants, and provides graphical insights to help users analyze air quality trends and their potential health impacts.

# Features
📊 Real-time AQI monitoring
🏙️ Data for top Indian cities
📅 Date-wise AQI tracking
🌫️ Major pollutants tracking (PM2.5, PM10, Ozone)
📈 Graphical visualization of AQI trends
🎯 User-friendly and interactive interface
🛠️ Tech Stack
